Angular2

Nemus Plant (내부 프로젝트)

| 네무스텍


1) 진행기간

  • 2016.11 ~ 2017.01

2) 주요내용

공장 내에 비콘을 설치하여 직원들의 안전을 관리하고 차량의 움직임을 파악할 수 있는 솔루션을 개발하기 위해 팀 내에서 진행한 프로젝트입니다. 이전에 진행했던 스마트 플랜트 사업을 내재화하는 작업을 진행하였고, Spring framework 기반의 웹 화면을 아래 기술들을 이용하여 바꾸는 작업을 진행했습니다.

3) 본인의 역할

  • 웹 기반 공장 내 비콘 및 차량 관리자 시스템 구축

4) 사용한 Skill 또는 지식

  • Angular2
  • MongoDB
  • Hapi.js
  • primeNG UI 라이브러리
  • npm
  • bootstrap
  • Chrome 개발자 도구
  • Gitlab
  • Redmine

5) 결과/성과

  • Angular2 개발 및 다양한 웹 기술 경험
  • REST API 구축
  • 사내 기획팀과의 협업 및 기획에 따른 기술검토
